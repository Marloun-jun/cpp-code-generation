## 🎯 Основная рабочая модель

| Модель | Папка | Описание | Статус |
|--------|-------|----------|--------|
| **10000** | `bpe_10000/` | **Основная модель для повседневного использования** | ✅ Рекомендуется |

Модель с размером словаря **10000 токенов** выбрана как основная, по результатам сравнения моделей Python-реализации токенизатора

## 📊 Модели для сравнения и бенчмарков

| Размер | Папка | Назначение |
|--------|-------|------------|
| **8000** | `bpe_8000/` | Для бенчмарков и сравнения производительности |
| **12000** | `bpe_12000/` | Для бенчмарков и сравнения производительности |

Эти модели **не предназначены для повседневного использования**, а служат для:
- 📈 **Бенчмарков** - сравнение скорости разных размеров словаря
- 📊 **Анализа** - изучение влияния размера словаря на качество токенизации
- 🔬 **Исследований** - выбор оптимального размера для конкретных задач

## 🔧 Использование в коде

### Базовый токенизатор (BPETokenizer)

**Для основной модели (рекомендуется):**
```cpp
#include "bpe_tokenizer.hpp"

bpe::BPETokenizer tokenizer;
tokenizer.set_byte_level(true);

// Загрузка основной модели 10000
if (tokenizer.load_from_files("models/bpe_10000/cpp_vocab.json", 
                              "models/bpe_10000/cpp_merges.txt")) {
    std::cout << "Основная модель загружена!" << std::endl;
}

// Кодирование текста
auto tokens = tokenizer.encode("int main() { return 0; }");
```
**Для бенчмарков (сравнение размеров):**
```cpp
// Модель 8000 для сравнения
tokenizer.load_from_files("models/bpe_8000/cpp_vocab.json", 
                          "models/bpe_8000/cpp_merges.txt");

// Модель 12000 для сравнения
tokenizer.load_from_files("models/bpe_12000/cpp_vocab.json", 
                          "models/bpe_12000/cpp_merges.txt");
```
### Оптимизированный токенизатор (FastBPETokenizer)
```cpp
#include "fast_tokenizer.hpp"

bpe::FastBPETokenizer tokenizer;

// Загрузка основной модели 10000
if (tokenizer.load("models/bpe_10000/cpp_vocab.json", 
                   "models/bpe_10000/cpp_merges.txt")) {
    std::cout << "Fast модель загружена!" << std::endl;
}

// Кодирование текста
auto tokens = tokenizer.encode("int main() { return 0; }");
```
## 🔄 Происхождение моделей

Все модели в этом каталоге являются конвертированными версиями из Python-реализации токенизатора. Исходные Python-модели находятся в:
```text
../../bpe_python/models/
├── bpe_8000/          # ➡️ models/bpe_8000/
├── bpe_10000/         # ➡️ models/bpe_10000/
└── bpe_12000/         # ➡️ models/bpe_12000/
```
Конвертация выполнена скриптом tools/convert_vocab.py:
```bash
# Конвертация всех моделей
python tools/convert_vocab.py --input ../bpe_python/models/bpe_8000 --output models/bpe_8000
python tools/convert_vocab.py --input ../bpe_python/models/bpe_10000 --output models/bpe_10000
python tools/convert_vocab.py --input ../bpe_python/models/bpe_12000 --output models/bpe_12000
```
