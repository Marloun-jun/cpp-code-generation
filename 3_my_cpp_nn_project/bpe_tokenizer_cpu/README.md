## 🎯 О проекте

**Цель:** Создание максимально быстрого и точного BPE токенизатора для обработки C++ кода с поддержкой русского языка и Unicode.

**Ключевые особенности:**
- ⚡ **Высокая производительность** — C++ версия обрабатывает **~22,000 примеров/сек** в реальном пайплайне
- 🚀 **SIMD оптимизации** — AVX2, AVX, SSE4.2 для максимальной скорости
- 🔄 **Параллельное обучение** — использование всех ядер CPU через OpenMP
- 🐍 **Полная Python интеграция** — обучение, валидация и использование из Python
- 🌐 **Веб-сервер** — REST API с Swagger UI и Prometheus метриками
- ✅ **Валидация** — полное сравнение C++ и Python реализаций
- 📊 **Бенчмарки** — сравнение с HuggingFace tokenizers

## 📊 Производительность

### Реальная производительность (DataLoader, batch=32)

Сравнение в одинаковых условиях (PyTorch DataLoader, 100 примеров C++ кода):

| Метрика | 🤗 HuggingFace | 🐍 Python BPE | ⚡ C++ BPE |
|---------|---------------|---------------|------------|
| Скорость обработки | ~2,100 экз/с | ~5,100 экз/с | **~22,300 экз/с** |
| Время encode (мкс) | ~470 | ~195 | **~16** |
| Время decode (мкс) | ~55 | ~193 | **~8** |
| Память (МБ) | 7.2 | 2.5 | **0.1** |
| Ускорение vs Python | - | 1.0x | **~4.3x** |

*Измерено на Intel i5-1240P, 16 ГБ ОЗУ, NVIDIA GTX 1650.*

### Пиковая производительность (микро-бенчмарк, модель 10000)

Чистое время выполнения одной операции `encode()` без накладных расходов:

| Метрика | 🤗 HuggingFace | 🐍 Python BPE | ⚡ C++ BPE |
|---------|---------------|---------------|------------|
| Время encode (мкс) | 0.462 | 0.197 | **0.007** |
| Время decode (мкс) | 0.045 | 0.185 | **0.003** |

**Пиковое ускорение C++ относительно Python: ~28x**  
**Пиковое ускорение C++ относительно HuggingFace: ~66x**

> **Примечание:** Пиковая производительность измеряет только алгоритм токенизации. Реальная производительность в пайплайне DataLoader составляет **~22,300 экз/с для C++** и **~5,100 экз/с для Python** (ускорение **~4.3x**), что отражает накладные расходы на загрузку данных, паддинг и батчинг.

## 🚀 Быстрый старт

### 1. Клонирование
```bash
git clone <repo_url>
cd bpe_tokenizer_cpu
```

### 2. Подготовка данных
```bash
cd data/scripts
python clean_dataset.py --input ../raw/2_cpp_code_generation_dataset.csv
python analyze_dataset.py
```

### 3. Обучение Python модели
```bash
cd ../../bpe_python
python trainer.py --vocab-size 10000
```

### 4. Конвертация в C++ формат
```bash
cd ../bpe_cpp/tools
python convert_vocab.py --model-size 10000
```

### 5. Сборка C++ токенизатора
```bash
cd ..
mkdir build && cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
make -j$(nproc)
```

### 6. Запуск тестов
```bash
# C++ тесты
cd bpe_cpp/build
ctest -V

# Python тесты
cd ../../bpe_python
python tests/test_bpe_tokenizer.py
```

### 7. Сравнение производительности
```bash
cd ../scripts
python benchmark_all.py
python plot_results.py
```

## 🤝 Интеграция

### Python

```python
from bpe_python.tokenizer import BPETokenizer

tokenizer = BPETokenizer.load('models/bpe_10000/vocab.json', 
                               'models/bpe_10000/merges.txt')
tokens = tokenizer.encode("int main() { return 0; }")
```

### C++

```cpp
#include "fast_tokenizer.hpp"

bpe::FastBPETokenizer tokenizer;
tokenizer.load("models/bpe_10000/cpp_vocab.json", 
               "models/bpe_10000/cpp_merges.txt");
auto tokens = tokenizer.encode("int main() { return 0; }");
```

### PyTorch

```python
from bpe_python.pytorch_integration import BPETokenizerWrapper, create_dataloader

wrapper = BPETokenizerWrapper(tokenizer, max_length=512)
dataloader = create_dataloader(texts, wrapper, batch_size=32)
```
