## 📂 Корпус C++ кода (`corpus/`)

Эта папка содержит подготовленные тексты программ на C++ для обучения и тестирования.

### **Файлы**

| Файл | Размер | Количество | Назначение |
|------|--------|------------|------------|
| `corpus.txt` | ~11.7 MB | 9322 строк | Полный корпус (все программы) |
| `train_code.txt` | ~10.0 MB | 8000 строк | Обучающая выборка |
| `test_code.txt` | ~0.6 MB | 500 строк | Тестовая выборка |
| `val_code.txt` | ~1.1 MB | 800 строк | Валидационная выборка |

### **Статистика**

- **Всего примеров:** 9322 программ
- **Размер полного корпуса:** ~11.7 MB
- **Средняя длина программы:** 1194 символа
- **Медианная длина:** 580 символов
- **Мин/Макс длина:** 82 / 27737 символов
- **Уникальных символов:** 243
- **Формат:** одна программа на одну строку (все переносы строк заменены на "\n")

## 📦 Сырые данные (`raw/`)

2_cpp_code_generation_dataset.csv — исходный файл в формате CSV, содержащий:

- Поле 1 - "description" с запросом
- Поле 2 - "code" с кодом программ

Этот файл является бэкапом и не используется напрямую в проекте. Для работы используйте подготовленные файлы из corpus/.

## 🔄 Происхождение данных

Данные были подготовлены с помощью скрипта scripts/clean_dataset.py:

- Исходные данные: CSV-файл с наличием поля "code"
- Очистка: удаление пустых строк, строк с комментариями и полей кроме поля 2 ("code")
- Разделение: 8000 train / 500 test / 800 val
- Сохранение: отдельные текстовые файлы

## 🔧 Как подготовить свои данные

### 1. Подготовка исходных данных

Создайте CSV-файл с полями (как в raw/2_cpp_code_generation_dataset.csv). Минимально необходимо поле "code" с кодом программы. Остальные поля опциональны.
```csv
"программа которая выводит простое приветствие без запроса ввода без using namespace std","#include <iostream>\n\nint main() {\n std::cout << \"Welcome to the program!\" << std::endl;\n return 0;\n}","explicit_std","basic_io","std::cout,std::endl"
```

### 2. Очистка датасета
```bash
cd scripts
python clean_dataset.py --input ../raw/your_data.csv --output-dir ../corpus
```
**Параметры скрипта:**

- --input - путь к входному CSV-файлу
- --output-dir - директория для сохранения результатов
- --train-size - размер обучающей выборки (по умолчанию 8000)
- --test-size - размер тестовой выборки (по умолчанию 500)
- --val-size - размер валидационной выборки (по умолчанию 800)

**Скрипт создаст:**

- corpus.txt        — все программы (мой dataset объемом 9300 примеров)
- test_code.txt     — программы для тестирования (для моего dataset 500 примеров)
- val_code.txt      — программы для валидации (для моего dataset 800 примеров)
- train_code.txt    — программы для обучения (оставшиеся 8000 примеров)


### 3. Анализ данных
```bash
cd scripts
python analyze_dataset.py
```

Скрипт автоматически найдет корпус в ../corpus/corpus.txt и создаст:

- Статистику в ../metadata/dataset_stats.json
- Графики в ../reports/figures/
- Отчет в ../reports/dataset_report.md

**Пример вывода:**
```text

============================================================

- Всего символов: 11,134,224
```

## 🎯 Использование в проекте

### Для обучения Python токенизатора
```python
from tokenizer import BPETokenizer

# Загрузка корпуса
with open('data/corpus/train_code.txt', 'r') as f:
    corpus = [line.strip() for line in f if line.strip()]

# Обучение
tokenizer = BPETokenizer(vocab_size=8000, byte_level=True)
tokenizer.train(corpus)
```

### Для обучения C++ токенизатора
```bash
# Скрипт обучения автоматически найдет данные в data/corpus/
cd ../bpe_cpp
python scripts/train_example.py --train ../data/corpus/train_code.txt
```

### Для тестирования
```python
from tokenizer import BPETokenizer

# Загрузка тестовых данных
with open('data/corpus/test_code.txt', 'r') as f:
    test_data = [line.strip() for line in f if line.strip()]

# Загрузка обученной модели
tokenizer = BPETokenizer.load('models/bpe_8000/vocab.json', 'models/bpe_8000/merges.txt')

# Тестирование
for code in test_data[:10]:
    tokens = tokenizer.encode(code)
    decoded = tokenizer.decode(tokens)
    print(f"Оригинал: {code[:50]}...")
    print(f"Токенов: {len(tokens)}")
    print(f"Совпадает: {code == decoded}")
```
