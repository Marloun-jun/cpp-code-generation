## 🎯 О проекте

**Цель:** Создание модели для генерации C++ кода по русским инструкциям.

**Особенности:**
- 🏗️ Архитектура Transformer (GPT-style)
- 🇷🇺 "Понимание" русских инструкций (через LoRA)
- 🔤 Русский BPE токенизатор (4000 токенов)
- 🌐 Веб-сервер для демонстрации
- 📊 Полный цикл: данные → обучение → тесты → демонстрация

## 📊 Модели

### Сравнение моделей

| Модель | Параметры | d_model | nhead | layers | Val Loss | Размер |
|--------|-----------|---------|-------|--------|----------|--------|
| **Tiny** | 5.2M | 192 | 3 | 3 | 1.68 | ~64.7 МБ |
| **Small** | 8.3M | 256 | 4 | 4 | 1.41 | ~101.1 МБ |
| **Medium** | 18.3M | 384 | 6 | 6 | **0.83** | ~222.5 МБ |
| **Medium + LoRA** | 18.3M + 1.8M | 384 | 6 | 6 | **0.86** | ~(222.5 + 7.2) МБ |

### Medium + LoRA (рекомендуемая)

| Параметр | Значение |
|----------|----------|
| Базовая модель | Medium (18.3M) |
| LoRA параметры | 1.79M (7.7%) |
| Ранг (r) | 32 |
| Alpha | 64 |
| Эпох LoRA | 5 |
| Лучшая эпоха | 5 (val_loss=0.86) |
| Vocab size | 14000 (10001 C++ + 3999 русских) |


## 🚀 Быстрый старт

### 1. Установка

```bash
# Клонирование репозитория
git clone <repo_url>
cd 3_my_cpp_nn_project

# Создание виртуального окружения
python -m venv .venv
source .venv/bin/activate

# Установка зависимостей
pip install -r requirements.txt
```

### 2. Подготовка данных

```bash
# Токенизация C++ корпуса
python scripts/prepare_train_dataset.py

# Извлечение русских описаний
python scripts/description_tokenizator/extract_descriptions.py

# Обучение русского токенизатора
python scripts/description_tokenizator/train_rus_tokenizer.py

# Подготовка датасета для LoRA
python scripts/prepare_finetune_dataset.py
```

### 3. Обучение модели

```bash
# Обучение Medium (рекомендуется)
python scripts/train_medium.py

# Или Tiny/Small
python scripts/train_tiny.py
python scripts/train_small.py
```

### 4. LoRA дообучение

```bash
# Дообучение на русских инструкциях
python scripts/train_finetune_lora.py

# Продолжить с последней эпохи
python scripts/train_finetune_lora.py --auto-resume
```

### 5. Запуск демонстрации (веб-сервера)

```bash
python scripts/final_server.py
# Открыть http://localhost:8080
```

### Примеры генерации

**Инструкция:** `Напиши программу с функцией main`
```cpp
#include <iostream>
int main() {
    int x = 10;
    int* p = &x;
    std::cout << *p << std::endl;
    return 0;
}
```

**Инструкция:** `Напиши программу которая выводит Hello World`
```cpp
#include <iostream>
#include <string>
#include <algorithm>
using namespace std;

int main() {
    string str = "hello world";
    cout << "Исходный: " << str << endl;
    return 0;
}
```
**Примечание:** Выбраны лучшие 2 из многих тестовых генераций

## ⚠️ Ограничения

- Размер датасета: ~9300 примеров — мало для production
- Качество генерации: Синтаксис не всегда корректный, семантика ограничена
- Понимание инструкций: Не всегда понимает запрос
- Рекомендуемый датасет: 200K+ примеров для хорошего качества
