## 📝 Исходные данные

### **`raw/2_cpp_code_generation_dataset.csv`** — исходный датасет

**Назначение:** Оригинальный CSV файл с полным датасетом для генерации C++ кода.

**Структура:**

| Поле | Описание | Пример |
|------|----------|--------|
| `description` | Русское описание задачи | `Напиши программу которая выводит Hello World` |
| `code` | C++ код | `#include <iostream>\nint main() {...}` |
| `style` | Стиль кода | `using_namespace_std / explicit_std` |
| `topic` | Тема | `basic_io, classes, functions` |
| `keywords` | Ключевые слова | `cout, class, vector` |

## 💻 Корпус C++ кода

### **`corpus/`** — текстовые данные

**Назначение:** Текстовые файлы с C++ кодом (по одной программе на строку) для обучения токенизатора и базовой модели.

| Файл | Записей | Назначение |
|------|---------|------------|
| `corpus.txt` | 9300 | Полный корпус (все программы) |
| `test_code.txt` | 500 | Тестовая выборка (5.4%) |
| `train_code.txt` | 8000 | Тренировочная выборка (86%) |
| `val_code.txt` | 800 | Валидационная выборка (8.6%) |

**Формат:** Каждая строка — отдельная C++ программа.

**Пример строки:**

```text
#include <iostream>\nusing namespace std;\n\nint main() {\n cout << \"Hello World\";\n return 0;\n}
```

**Использование:** Обучение базовой модели Medium (только код)

## 🔢 Токенизированные данные

### **`tokenized/`** — готовые тензоры

**Назначение:** Предварительно токенизированные данные в формате PyTorch (.pt) для быстрой загрузки при обучении.

| Файл | Примеров | Размер | Формат |
|------|----------|--------|--------|
| `test_tokens.pt` | 500 | ~3.2 МБ | List[Tensor] |
| `train_tokens.pt` | 8000 | ~48.4 МБ | List[Tensor] |
| `val_tokens.pt` | 800 |  ~5.4 МБ | List[Tensor] |

**Структура:** Список тензоров, где каждый тензор — последовательность ID токенов.

**Токенизатор:** C++ BPE, vocab_size = 10001

**Специальные токены:**

| Токен | ID | Назначение |
|-------|----|------------|
| `<PAD>` | 0 | Паддинг |
| `<UNK>` | 1 | Неизвестный токен |
| `<BOS>` | 2 | Начало последовательности |
| `<EOS>` | 3 | Конец последовательности |

**Загрузка:**
```python
import torch

train_tokens = torch.load('data/tokenized/train_tokens.pt')
print(f"Загружено {len(train_tokens)} примеров")
print(f"Первый пример: {len(train_tokens[0])} токенов")
```

## 🎯 Данные для LoRA (инструкции)

### **`instruction_train.jsonl`** — инструкции + код

**Назначение:** Датасет для дообучения LoRA на понимание русских инструкций.

**Формат:** JSON Lines (.jsonl), каждая строка — отдельный JSON объект.

**Структура:**
```json
{"text": "напиши программу которая отображает 'Hello World' без перехода на новую строку\n\n#include <iostream>\nusing namespace std;\n\nint main() {\n cout << \"Hello World\";\n return 0;\n}"}
```

**Статистика:**

- Всего примеров: 20 540
- Формат: `[Инструкция]\n\n[Код]`
- Размер: ~24,0 МБ

**Создание:**
```bash
python scripts/prepare_finetune_dataset.py
```

**Загрузка:**
```python
import json

data = []
with open('data/instruction_train.jsonl', 'r') as f:
    for line in f:
        data.append(json.loads(line))

print(f"Загружено {len(data)} примеров")
```

### **`rus_descriptions.txt`** — русские описания

**Назначение:** Текстовый файл с русскими описаниями для обучения русского BPE токенизатора.

**Статистика:**

- Строк: 9300
- Уникальных слов: ~2500
- Размер: ~1.5 МБ

**Использование:**
```bash
# Обучение русского токенизатора (4000 токенов)
python scripts/description_tokenizator/train_rus_tokenizer.py
```
