# C++ BPE Tokenizer

Высокопроизводительная реализация Byte Pair Encoding (BPE) токенизатора, оптимизированная для кода на C++. 
Проект включает как базовую версию для валидации, так и оптимизированную версию с SIMD, пулами памяти и параллельным обучением.

## Особенности

- **Две реализации**: базовая (для валидации) и оптимизированная (FastTokenizer)
- **SIMD оптимизации**: AVX2, SSE4.2 для массовых операций
- **Параллельное обучение**: OpenMP, многопоточный подсчет частот
- **Кэширование**: LRU-кэш для повторяющихся входных данных
- **Пул памяти**: уменьшение количества аллокаций
- **Python биндинги**: через pybind11
- **Веб-сервер**: REST API на базе CrowCpp
- **Подробная статистика**: профилирование, метрики производительности

## Требования

- CMake 3.16+
- Компилятор C++17 (GCC 9+, Clang 10+, MSVC 2019+)
- Python 3.8+ (для биндингов и скриптов)
- OpenMP (опционально, для параллельного обучения)
- Git (для загрузки зависимостей)
